# 學生登入簽到

{% hint style="info" %}
此頁面建立中
{% endhint %}

## 一、點選老師張貼的簽到連結

老師在上課前會將「簽到連結」張貼在 Google Classroom 公告。

![](https://i.imgur.com/9U1E01k.png)

## 二、登入學校 Google 帳號

看到此畫面時，請點選自己的學校，接著使用學校的 Google 帳號登入。

第一次出現授權視窗則點按「允許」即可。

![](https://i.imgur.com/tG06MfQ.png)


## 三、點選「簽到」

登入後，會出現當節的簽到資訊，畫面中出現當下的時間點，即代表簽到成功。

![](https://i.imgur.com/Elst9Xp.png)
